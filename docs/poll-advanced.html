Polls - Advanced Use Case

Password Protected Polls

By default this field is empty. If you need to create a password protected Poll then you may enter your desired password in the Password field and click the save button. Now whenever any participant tries to vote will be asked for a password.

Time Limited Polls

You can also have the poll enabled at a specific date and time that you select. For this you will have to fill the Link Active From Date and / or Link Active Till Date. Once set the Poll will be available between the selected date and time.

Multi Answer Polls

If you check the Checkbox labeled Enable Multiple Answers then the participants will be able to select multiple answers in their response.
